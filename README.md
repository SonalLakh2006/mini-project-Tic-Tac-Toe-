# Tic Tac Toe Game

## Project Overview
This project is a simple console-based Tic Tac Toe game developed using C++. The game demonstrates the use of arrays, loops, conditional statements, and basic game logic.

## Aim
To develop a Tic Tac Toe game in C++ that allows two players to play interactively through the console.

## Features
- Dynamic game board display
- Two-player gameplay
- Win detection
- Draw detection
- Replay option
- Uses arrays and loops
- Console-based interface

## Technologies Used
- C++
- Visual Studio Code
- GCC Compiler

## Project Structure
├── game.cpp
├── README.md
└── game project report.pptx

## How to Run
1. Clone the repository.
2. Compile the program:
   g++ game.cpp -o game
3. Run:
   ./game

## Game Rules
- Player 1 uses X.
- Player 2 uses O.
- The first player to get three marks in a row wins.
- If all cells are filled without a winner, the game is a draw.
